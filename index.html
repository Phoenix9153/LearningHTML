What is HTML?

HTML is the standard markup language for creating Web pages.

  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of Web pages using markup
  • HTML elements are the building blocks of HTML pages
  • HTML elements are represented by tags
  • HTML tags label pieces of content such as "heading", "paragraph", "table", and so on
  • Browsers do not display the HTML tags, but use them to render the content of the page

Creating your personal HTML document

Steps to set up a HTML document:

  1. Open Notepad for PC or Textedit for Mac
  2. For Textedit, make sure in Preferences that plain text format is selected. Afterwards, go to the Open and Save tab and tick the box saying "Display HTML files as HTML code instead of formatted text" or something similar.
  3. Write any HTML you wish into the document
  4. Save the document as a HTML file with the .html extension and set the encoding to UTF-8 (which is the preferred encoding for HTML files).
  5. To make a stylesheet, repeat similar steps except save the file with a .css extension.
  6. View your html page in your browser by double clicking the file or by right-clicking and select "Open with…"

Starting Code

As stated previously, HTML elements are represented by tags. Usually, these tags have a start tag and a corresponding tag, for example:

  <tagname>Content goes here…</tagname>

The HTML element is everything from the start tag to the end tag

First of all, HTML documents always start with <!DOCTYPE html>. This tag declares the document to be HTML5 and only needs to be stated once at the beginning of the document.

The HTML document itself begins with <html> and ends with </html>.
